# MineJason.Extras

MineJason is a suite of libraries that deals with various formats used in Minecraft: Java Editions.

This is the "Extras" module, which contains stuff that does not fit into the Client module.

For more information, go to the [GitHub repository](https://github.com/WithLithum/MineJason).
