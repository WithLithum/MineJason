# MineJason.Serialization.fNbt

This is a library that provides fNbt format support to
`MineJason.Serialization` library.

## Usage

Direct consumption of the types provided by this library requires a separate
article, which have yet been writing due to the library being in early
development.

## Contributing

To report issues or contribute code for this library, go to the
[project repository](https://codeberg.org/MineJason/serialization). See the
README file for more information.
